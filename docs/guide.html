◐ Kakudo Synth 使い方ガイド

スマホを傾けて音を奏でる PWA シンセサイザー。

1. インストール（ホーム画面に追加）

  • iPhone (Safari): 共有ボタン → 「ホーム画面に追加」
  • Android (Chrome): メニュー → 「アプリをインストール」／「ホーム画面に追加」

ホーム画面から起動すると全画面のアプリとして動きます。オフラインでも起動できます。

2. 演奏する

  1. 「タップして開始」を押す（iPhone はここでセンサー利用の許可ダイアログが出ます → 許可）。
  2. スマホを画面が下・地面と平行になるように構える。
  3. 左右に傾ける → 音程が変わる。
  4. 前後に傾ける → 明るさ（音色の抜け）が変わる。
  5. 「停止／再生」ボタンで音の on/off。

傾きセンサーが無い端末（PC など）では、中央のパッドを指やマウスでドラッグして演奏できます。

3. コントロール一覧

音色6 種類のシンセ音（サイン / ウォーム / スクエア / ノコギリ / スーパーソウ / FMベル）から選択。
感度少しの傾きでどれだけ大きく音が変わるか。大きいほど敏感。
変化の速さ傾きに音が追従する速さ。「遅い」ほど滑らかにグライド、「速い」ほど機敏。
音域(半音)音程が動く幅。小さいほど狭い音域で繊細に、大きいほど大胆に。
明るさ幅前後の傾きで開くローパスフィルターの上限。大きいほど明るい音まで出る。
音量マスター音量。
音階に合わせるONで音程を選んだ音階の音にスナップ。OFFで連続的に変化。
音階メジャー / マイナー / ペンタトニック / ブルース / 半音階 / 全音音階。
今の傾きを中央にセット現在の構え方をニュートラル（中央）として校正。楽な姿勢を基準にできます。

4. うまく鳴らないとき

  • 音が出ない → まず「タップして開始」→「再生」を押したか確認。マナーモード/音量も確認。
  • iPhone で傾きが効かない → センサー許可を拒否した可能性。アプリを再読み込みして許可し直す。
  • 傾きの基準がずれる → 「今の傾きを中央にセット」を押す。
  • センサーは https または localhost でのみ動作します。
