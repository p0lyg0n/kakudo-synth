◐ Kakudo Synth 使い方ガイド

スマホを傾けて音を奏でる PWA シンセサイザー。

1. インストール（ホーム画面に追加）

  • iPhone (Safari): 共有ボタン → 「ホーム画面に追加」
  • Android (Chrome): メニュー → 「アプリをインストール」／「ホーム画面に追加」

ホーム画面から起動すると全画面のアプリとして動きます。オフラインでも起動できます。

2. 演奏する

  1. 「タップして開始」を押す（iPhone はここで傾き・モーション利用の許可ダイアログが出ます → 許可）。
  2. スマホを画面が下・地面と平行になるように構える。
  3. モードに応じて傾けると音が変わる（下記）。
  4. 画面下の「停止／再生」ボタンで音の on/off。
  5. 設定は右上の⚙ボタンから開きます（画面はスクロールしません）。

傾きセンサーが無い端末（PC など）では、中央のパッドを指やマウスでドラッグして演奏できます。

2.5 モード

軸モード左右に傾ける → 音程、前後に傾ける → 明るさ。X-Y の2軸で操作します。
中心モード中心（水平）で低い音が鳴り、どの方向でも中心から離れるほど音が高くなります。離れるほど明るくもなります。

2.6 振動でジャンプ音（びよよーん）

設定の「振動エフェクト」を ON にすると、スマホを強く振った瞬間にバネのようなジャンプ音「びよよーん」が鳴ります。感度よく反応します。不要なら OFF にできます。

2.7 設定の保存 / 共有

  • モード・音色・各スライダー等の設定は自動的に記憶され、次回起動時に復元されます。
  • 設定内の「共有する」「リンクをコピー」ボタン、またはQRコードでアプリを他の人に渡せます。

3. コントロール一覧

演奏モード軸モード / 中心モードの切替。すぐ下に「中心にする」ボタン。
音色6 種類のシンセ音（サイン / ウォーム / スクエア / ノコギリ / スーパーソウ / FMベル）から選択。
音の変化「無段階（連続・スライド）」または各音階（メジャー / マイナー / ペンタトニック / ブルース / 半音階 / 全音音階）。音階を選ぶとその音にスナップ。
基準キー全体を移調する基準の音（A〜G#）。
高さの幅音程が動く幅。せまい↔ひろい。
効き（感度）少しの傾きでどれだけ変わるか。ゆるやか↔敏感。
音の追従傾きへの追従の速さ。なめらか↔機敏。
明るさ前後の傾きで開くフィルター上限。暗い↔明るい。
ゆらぎ（ビブラート）音程の細かな揺れ。なし↔強い。
残響（エコー）やまびこ状の残響。なし↔広い。
音量マスター音量。
振動エフェクト振ると「びよよーん」のON/OFFと、その感度。
中心にする現在の構え方をニュートラル（中心）として校正。楽な姿勢を基準にできます。

4. うまく鳴らないとき

  • 音が出ない → まず「タップして開始」→「再生」を押したか確認。マナーモード/音量も確認。
  • iPhone で傾きが効かない → センサー許可を拒否した可能性。アプリを再読み込みして許可し直す。
  • 傾きの基準がずれる → 「今の傾きを中央にセット」を押す。
  • センサーは https または localhost でのみ動作します。
